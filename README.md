## Local Development

To run this project locally, ensure you have Node.js and npm installed.

**1. Clone the repository**
```bash 
git clone [https://github.com/your-username/sripad-portfolio.git](https://github.com/sripadsanand/sripad-portfolio.git) 
cd sripad-portfolio 
2. Install dependencies

Bash
npm i 
3. Start the development server

Bash
npm run dev 
The local server will start with auto-reloading and an instant preview.
